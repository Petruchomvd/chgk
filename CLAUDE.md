# CHGK Project — Инструкции для Claude

## Сохранение планов реализации

Когда ты входишь в plan mode и составляешь план реализации задачи:

1. **Сохрани план** как файл в `docs/plans/` с именем `YYYY-MM-DD-<краткое-описание>.md`
   - Пример: `docs/plans/2026-02-28-parallel-classification.md`
2. **Формат файла:**
   ```markdown
   # <Название задачи>

   **Дата:** YYYY-MM-DD
   **Статус:** В работе | Выполнен | Отложен

   ## Контекст
   <Зачем это нужно>

   ## План
   <Шаги реализации — то, что было в plan mode>

   ## Затронутые файлы
   <Список файлов, которые будут изменены>

   ## Результат
   <Заполняется после реализации>
   ```
3. **Сохраняй план ДО начала реализации** (сразу после выхода из plan mode)
4. **Обнови статус и раздел "Результат"** после завершения работы

Папка `docs/plans/` — архив всех планов реализации. Это помогает отслеживать историю решений и причины изменений в проекте.
