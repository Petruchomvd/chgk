# Ручная проверка спорных вопросов qwen-2.5-72b-instruct

**Дата:** 2026-03-07
**Статус:** В работе

## Контекст

После прогона 2774 вопросов через qwen/qwen-2.5-72b-instruct (OpenRouter, 2-stage) нужно оценить качество классификации и найти системные ошибки модели для улучшения промпта.

## Методология

1. Создан скрипт `scripts/review_disputed.py` — автоматический отбор спорных вопросов
2. Критерии спорности (dispute_score):
   - Низкая уверенность primary (<0.6 → +3, <0.7 → +1)
   - Близкие confidence двух категорий (diff <0.05 → +3, <0.1 → +2)
   - Confusion-пары (известные проблемные пары категорий) → +2
   - Логика как primary → +1
3. Проверены 150 вопросов из 3 категорий (по 50)
4. Оценка top-2: если правильная категория в primary ИЛИ secondary — засчитывается

## Результаты ручной проверки

### Логика и wordplay: 39/50 (78%) верно

Основные ошибки:
- Модель ставит "Логика" когда есть элемент wordplay, но основная тема — конкретная предметная область
- Примеры: вопросы о конкретных исторических фактах с игрой слов → должна быть История, не Логика

### Кино и театр: 36/50 (72%) верно

Основные ошибки:
- Модель ставит "Кино" когда упоминается фильм/актёр, но вопрос по сути о другой области
- Литература → Кино (экранизации: если вопрос о книге, не о фильме)
- Искусство → Кино (если вопрос о живописи/архитектуре, упомянутой в фильме)

### Быт и повседневность: 22/50 (44%) верно — ХУДШАЯ категория

Основные ошибки:
- Модель классифицирует по поверхностным объектам (ручка, зажигалка, машина → "Быт")
- Реальная утечка:
  - **История** (9 вопросов): исторические события с бытовыми предметами
  - **Общество и политика** (5+): политический/социальный контекст
  - **Логика и wordplay** (4+): игра слов вокруг бытовых предметов
  - **Наука** (3+): научные факты о бытовых предметах

## Выводы и рекомендации для промпта

### 1. Быт — главная проблема (44% accuracy)
- Добавить правило: бытовой предмет ≠ категория "Быт". Важен контекст: если вопрос об истории предмета → История, о научном принципе → Наука
- Добавить few-shot примеры confusion-пар Быт↔История, Быт↔Общество

### 2. Кино — поверхностная классификация (72%)
- Усилить правило: упоминание фильма/актёра не делает вопрос "Кино". Если вопрос о литературном произведении → Литература
- Добавить примеры экранизаций → Литература

### 3. Логика — приемлемо, но можно улучшить (78%)
- Ужесточить: wordplay как форма, а не тема. Если для ответа нужны конкретные знания → предметная категория
