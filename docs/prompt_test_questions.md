## Итого

- **72B**: 5/10 (50%)
- **14B**: 2/10 (20%)

## Паттерны ошибок

### 72B
- Цепляется за поверхностные признаки: город → География, "разборки" → Спорт
- Военный контекст с бытовыми словами → ошибочно Быт
- Wordplay не распознаёт

### 14B
- Систематически сваливает в "Быт и повседневность" при любых бытовых словах
- Не различает исторический контекст

## Предложения по промпту
- Усилить: "Если речь о военном флоте, армии, сражениях — это История → Военная история, даже если упоминаются бытовые предметы"
- Добавить примеры пограничных случаев в few-shot
